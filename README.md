# Rogan-Helper (老眼OCRルーペ)

This is a small personal Web app created to help read small printed text
(e.g. labels, manuals, product information) that becomes difficult to see with age.

The app is intentionally simple and lightweight.
It is designed for personal use, not as a commercial service.

これは、年齢とともに読みにくくなる小さな印刷文字（例：ラベル、マニュアル、製品情報）の読み取りを支援するために作成された、個人向けの小さなWebアプリです。

このアプリは意図的にシンプルで軽量に設計されています。商用サービスではなく、個人での使用を目的として設計されています。

## Philosophy

This project is not about technical perfection.
It is about choosing what is *enough*.

- No continuous OCR
- No heavy processing
- No unnecessary features

Just read, and leave a small trace if needed.
